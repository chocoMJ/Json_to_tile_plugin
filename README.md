## Json to Tile plugin
### 1. 프로젝트 구성
SketchToJson 프로젝트는, 사용자의 손그림 이미지 입력을 받아 이를 타일 맵으로 변환하는 프로젝트이다. 사용자가 이미지를 입력시키면, 웹 에디터상에서 수정을 거치고 최종적으로 타일맵 정보를 담은 Json 파일을 획득할 수 있다. 본 레포지토리는, 획득한 Json파일을 언리얼 에셋과 매핑하여 실제 언리얼 타일 맵을 생성하는 플러그인이다.
